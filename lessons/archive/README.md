# Lesson Archive

This folder holds old lesson previews, style galleries, and palette experiments that should not appear in the active lesson list.

Active course lessons stay directly under `lessons/` and should link:

```html
<link rel="stylesheet" href="../assets/style.css">
<script src="../assets/quiz.js"></script>
```

Archived pages may point to styles under `../../assets/archive/` so the historical previews can still be opened if needed.
